1.github地址:https://github.com/Yangtark/tugonet

2.首页目录：index.html

3.项目文件里面的sass文件

4.模块化主文件目录是js文件夹里的check.js

5.项目需要在Apache环境下运行

6.注册登录可以点击,首页选项卡可以点击,首页楼层可以点击,购物车也可以点击,
登录成功之后可以点击退出登录,商品列表可以点击商品进入商品详情页

7.项目说明：当用户没登录之前点击购物车会进入登录页面,这时用户需要注册,注
册成功之后可以进入购物车,点击商品可以添加购物车,当用户退出登录之后,再次
点击购物车将会回到登录注册页面,点击商品列表的每一个商品会进入到所对应的
商品详情页,商品详情页的商品图片有放大镜效果,并且商品详情页只有一个html,
点击的对应商品渲染的只是商品详情页的内容,商品列表是由ajax渲染出来的,
购物车也是由ajax渲染出来的,点击不同的商品购物车都会出现此商品信息。
